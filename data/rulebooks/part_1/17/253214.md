## Comprehensive Reference Manual

### 1. Lore & Objective  
Escape Tales: The Awakening is a cooperative narrative-driven escape room card game for 1–4 players. You play as Samuel, a father on a desperate journey to find and save his daughter, Lizzy. The story unfolds through immersive narrative and puzzle-solving, with decisions directly influencing the path and ending of the game. The primary objective is to locate Lizzy by exploring locations, solving puzzles, and making meaningful choices. The game features multiple endings—some happy, others tragic—determined by player actions.

---

### 2. Components  
- **18 Action tokens**  
- **122 Game cards (C001–C122)**  
  - Includes puzzle, item, exit, and condition cards  
- **9 Doom cards (D0–D8)**  
- **18 Location cards (L01–L18)**  
- **1 Game board**  
- **1 Story Book**  
- **Digital app access** (https://app.escape-tales.com)

---

### 3. Setup  
1. Place the **game board** centrally so all players can reach it.  
2. Prepare the following components in accessible areas:  
   - **A**: 18 Action tokens  
   - **B**: Game cards, sorted in ascending order (C001 to C122)  
   - **C**: Doom cards, sorted in ascending order (D0 to D8)  
   - **D**: Location cards, sorted in ascending order (L01 to L18)  
   - **E**: Story Book  
3. Ensure all cards are placed with **back side up**—never check the front during setup.  
4. Prepare a mobile device or computer to access the **Escape Tales app** (https://app.escape-tales.com).  
5. Verify all components are present by checking card counts. Contact support if anything is missing: biuro@boardanddice.com

### 4. Gameplay Flow  
The game progresses in a loop of reading, exploring, and solving:

1. **Read the Story Book Paragraph**  
   - Begin with the prologue on the front cover.  
   - Each paragraph is labeled (e.g., P854) and contains:  
     - **A**: Paragraph number  
     - **B**: Story text (read first)  
     - **C**: Action(s) to perform (follow in order)  
   - Only read the indicated paragraph—do not preview others on the same page.  

2. **Perform Actions Until Directed Back to the Story Book**  
   Players may perform any of the following:  
   - **Exploration**:  
     - Place one **action token** on an unexplored area of the current **map card**.  
     - Read the paragraph number linked to that area.  
   - **Doom (when out of action tokens)**:  
     - If no action tokens remain and you wish to explore, draw the top **Doom card**.  
     - Read the story and resolve all effects listed.  
     - You cannot draw a Doom card if action tokens are still available.  
   - **Solving Puzzles**:  
     - Open the corresponding puzzle in the app.  
     - Input answers, check required cards, or request hints.  
     - Solving a puzzle may unlock a new Story Book paragraph.  
   - **Leaving a Location**:  
     - If you possess an **exit card**, follow instructions to leave.  
     - Go to the specified Story Book paragraph.  
     - Once left, you cannot return to the location.  
   - **Fulfilling a Condition**:  
     - If you meet criteria on a **condition card**, go to the indicated paragraph.  
     - Conditions can only be resolved within the same location.  

3. **Repeat** from Step 1 whenever a component directs you back to the Story Book.

### 5. Core Mechanics  

#### Location Cards & Map Cards  
- At the start of a location, you receive:  
  - 2 Location cards  
  - 1 Map card  
  - Action tokens (as instructed)  
- **Map card** shows a 12-square grid matching the game board.  
- Place Location cards on the board according to the map card’s layout.  
- Each square is an **area** that can be explored.  
- Only explored areas (with action tokens) trigger story progression.  

#### Action Tokens  
- Represent your ability to explore.  
- Limited supply (18 total).  
- Once depleted, further exploration requires drawing a **Doom card**.  

#### Doom Cards  
- Draw only when:  
  - You have **no action tokens left**  
  - You want to explore a new area  
- Each Doom card has:  
  - **Top**: Narrative text  
  - **Bottom**: Game effects (e.g., lose an item, gain a card, story change)  
- Overuse may affect gameplay and endings.  

#### Game Cards (C001–C122)  
Types:  
- **Puzzle Cards**  
  - Red icon in bottom right.  
  - Belong to specific puzzles—group by symbol.  
  - All cards with the same symbol are required to solve the puzzle (check app for count).  
- **Item Cards**  
  - Gray background with item image.  
  - May have brown icon in bottom left.  
  - Kept in play area for future use.  
- **Exit Cards**  
  - No symbol; feature a black "passage" area in the center.  
  - Allow leaving the current location.  
- **Condition Cards**  
  - Enable special actions if criteria are met (e.g., having certain cards).  
  - Can only be resolved in the current location.  
- **Multi-type Cards**: A single card may belong to multiple categories.

#### App Integration  
- Access via: https://app.escape-tales.com  
- Works offline after first load.  
- Select language matching your game version.  
- App features:  
  - **Puzzle Menu**: One tab per puzzle symbol.  
  - **Answer Space**: Input solution (letters/numbers).  
    - Number of slots = length of answer.  
    - Correct answer redirects to a Story Book paragraph.  
  - **Required Cards**: Shows how many and which types of cards are needed.  
  - **Hints**:  
    - “Take a hint” → later becomes “See the answer”  
    - Use as last resort; no gameplay penalty.  
- App actions do **not** consume action tokens or affect game state negatively.

### 6. Scoring & End Game  
- There is **no scoring system**.  
- The game ends when a Story Book paragraph leads to a **final ending**.  
- Multiple endings exist—ranging from success to tragedy—based on:  
  - Puzzle solutions  
  - Card management  
  - Use of Doom cards  
  - Key decisions  
- The game is **non-competitive** and fully cooperative.  
- After finishing, you may replay to explore alternate paths.  
  - Recommended: Start from **P015** to skip the first location.

### 7. FAQ or Edge Cases  
- **Can I return to a previous location?**  
  No. Once you leave a location using an exit card, you cannot return.  

- **What if I solve a puzzle incorrectly in the app?**  
  You can retry as many times as needed. Only correct answers advance the story.  

- **Do I need internet during gameplay?**  
  No. The app works offline after initial page load.  

- **Can I use hints freely?**  
  Yes. Hints and answers in the app do not penalize gameplay or affect endings.  

- **What if I run out of Doom cards?**  
  The rules do not specify, but running out implies no further exploration is possible without tokens. Manage Doom use carefully.  

- **How do I save progress?**  
  Use the **save table** on the last page of the rulebook. Record:  
  - Current paragraph  
  - Cards in play  
  - Doom cards used  
  - Items held  
  - Puzzles solved  
  Recommended to save when leaving a location.
